### Prerequisites
In this section, you will find all the necessary steps that should be completed before running the application:

1. Download NodeJS to your machine (v 22.12.0 preferably)
2. Download any IDE of your choice
4. Clone the repo
   ```git clone https://github.com/RostykVoloshyn/Pw-Demo.git```
5. Install all the dependencies by running
```npm install --force ```

### How to start test execution
Before running the tests, you need to start the test application:
1. In the IDE Termainl navigate to ```pw-practice-app``` directory
2. Run the command to start the application ```npm start```
3. Use standart Playwright commands to trigger the test execution

NOTE:

All tests ale located in ```tests``` subfolder under ```pw-practice-app``` and ```PW-APITEST-APP``` folders
The test application is taken from the internet and is used for learning purposes
